<Note>**Archetype**: Generator (G) — DPA contract</Note>

## Vấn đề giải quyết

DN (controller) thuê vendor (processor) xử lý data — cần DPA bổ sung HĐ chính theo Luật PDPL 91/2025 + GDPR-like clauses cho cross-border.

## Sections chuẩn DPA

1. Định nghĩa controller / processor / sub-processor
2. Phạm vi xử lý data
3. Instructions từ controller
4. Bảo mật + breach notification
5. Sub-processor approval
6. Audit right
7. Cross-border transfer (Luật PDPL Đ.25 + GDPR Art. 46)
8. Return / delete data on termination

## Cite anchors

- **Luật PDPL 91/2025/QH15** Đ.18-29
- **NĐ 13/2023/NĐ-CP** Đ.27-29 (xử lý qua bên thứ ba)
- GDPR Art. 28 nếu EU data

## Hỗ trợ workflow

- Procurement vendor onboarding (SaaS, hosting)
- Cross-border data transfer with EU parent
- Customer-facing DPA template stack
