## Catalog là gì

Mỗi skill = **1 con robot pháp lý chuyên môn hẹp**, đóng gói: prompt + reference luật + schema input/output + 2 ví dụ smoke test. Bạn gọi tên skill, đưa case data, robot trả về artifact pháp lý (hợp đồng / báo cáo audit / đơn kiện / memo) **kèm citation theo điều luật Việt Nam**.
