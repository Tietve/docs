<Warning>
**Quy tắc số 1**: Mọi điều khoản canonical / verdict / claim pháp lý PHẢI có cite anchor format `{LUẬT} Đ.{số} K.{khoản} điểm {chữ}`. Hallucinate cite = skill FAIL.
</Warning>

## Format chuẩn

```
BLDS 2015 Đ.466 K.5
BLLĐ 2019 Đ.36 K.2
BLTTDS 2015 Đ.189
LDN 2020 Đ.148 K.1
LĐĐ 2024 Đ.27 K.1
Luật ATVSLĐ 2015 Đ.75
NĐ 132/2020/NĐ-CP Đ.5
TT 103/2014/TT-BTC Đ.13
```

## Cite verify workflow

<Steps>
  <Step title="Skill emit cite">
    Output JSON có field `cite_anchors: ["BLDS Đ.X K.Y", ...]`
  </Step>
  <Step title="Pre-flight check">
    Skill validate cite format đúng `{LUẬT} Đ.{số}` pattern.
  </Step>
  <Step title="Reference lookup">
    Cross-check với `references/cite-anchors.md` trong skill.
  </Step>
  <Step title="User verify (MANDATORY)">
    User đọc artifact và **TRA NGƯỢC** điều luật. Hallucinate phổ biến với điều xa (Đ.700+).
  </Step>
</Steps>
